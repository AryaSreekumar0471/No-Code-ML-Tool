SimpliML Workflow

The SimpliML pipeline guides you through every step of building a machine learning model for healthcare datasets without coding. The workflow starts with uploading a dataset, followed by preprocessing where you can remove duplicates or missing values, scale numeric features, encode categorical features, and handle outliers. Next, you select the target and features, then apply a train–test split to divide the data. Class balancing is applied only to the training set, and k-fold cross-validation can be configured to improve reliability. You then choose a model — Decision Tree, KNN, or Logistic Regression and proceed to training with visualizations. Depending on the model, the system generates feature importance, decision boundaries, precision–recall curves, and confusion matrices. A detailed Excel report of preprocessing, data splits, balancing, validation, and results can be downloaded, and you can experiment interactively with the trained model to test predictions.
